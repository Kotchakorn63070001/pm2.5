# swt-project

## Project setup
```
cd swt-project
npm install

npm i vue-chartjs

npm i axios

npm i chart.js

npm i vue-router

npm i cors
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
